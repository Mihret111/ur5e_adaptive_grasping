# ur5e_adaptive_grasping
Develop adaptive grasping behaviors for soft, deformable, or fragile objects using force feedback control with sim-to-real validation.
1) Set up UR5e + 2FG7 model in IsaacSim with ROS2 bridge for soft/fragile object manipulation
2) Implement multiple grasp strategies for deformable or delicate objects based on compliance and fragility
3) Use force feedback for adaptive grasping and safe force regulation in simulation and real robot
4) Create manipulation primitives such as soft-push, pull, slide, and delicate pick-and-place
5) Validate sim-to-real transfer on a variety of soft and fragile objects:
Software needed: IsaacSim, MoveIt2, ROS2 Humble, force control libraries, soft-body/contact simulation tools
Research needed: Soft grasping, impedance/force control, manipulation planning for fragile objects, sim-to-real transfer for deformable materials
Deliverables: IsaacSim setup for soft-object manipulation, adaptive grasping strategy library, sim-to-real validation report for soft/fragile objects
